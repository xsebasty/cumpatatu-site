# cumpatatu-site

Site-ul static al aplicatiei Cumpatatu (cumpatatu.ro): landing one-page +
politica de confidentialitate (RO/EN). HTML + CSS pur, zero JS, zero tracking,
zero dependinte. Stilul replica tokens din DESIGN_SYSTEM-ul aplicatiei.

Gazduit pe GitHub Pages. Capturile de ecran se activeaza scotand atributul
`hidden` al sectiunii `#capturi` din `index.html` cand pozele exista in
folderul `capturi/`.
